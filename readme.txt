Add some fun and excitement to your site with confetti effects on any page of your WordPress site. Premium version integrates automatically with popular e-commerce, form, payment, and LMS plugins!

== Description ==

https://www.youtube.com/watch?v=-leuiB9RmHM

A fun confetti effect you can add anywhere on your site via block or shortcode. Choose from one of six effect styles!

Shortcode: [confetti]

[Visit our website for more information](https://wpsunshine.com/plugins/confetti/?utm_source=wordpress.org&utm_medium=link&utm_campaign=confetti-readme)

**Upgrade to Premium**

== Installation ==

1. Upload and activate the plugin
2. Go to Settings > Confetti and select your confetti effect
3. Use the Confetti block OR shortcode to include on any page of your site
